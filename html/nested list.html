HTML ATTRIBUTES
  • All HTML elements can have attributes.
  • Attributes provide additional information about elements.
  • Attributes are always specified in the opening tag.
  • Attributes usually come in name/value pairs like: name="value".
  1. The Style Attribute
    2. The style attribute is used to add styles to an element, such as color, font, size, and more.
  2. The lang attribute
    3. You should always include the lang attribute inside the html tag, to declare the language of the Web page. This is meant to assist search engines and browsers.
  3. The Type Attribute
    4. For both OL and UL, the type attribute specifies rendering options for visual user agents.
    • For the UL element, possible values for the type attribute are disc,square, and circle.
    • For the OL element, it can be set to "1" for NUMERICAL,"A" for UPPERCASE alphabeticle,"a" for lowercase alphabeticle, "I" for UPPERCASE ROMAN numerals, "i" for lowercase Roman numerals.
